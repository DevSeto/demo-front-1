Birddesk documentation

Initial install

  1. sudo apt-get install nodejs
  2. sudo apt-get install npm
  3. sudo ln -s /usr/bin/nodejs /usr/bin/node
  4. npm install -g bower
  5. npm install -g typings

Install

$ git clone https://{{Username}}@bitbucket.org/jantonk/birddesk-spa-frontend.git
$ cd birddesk-spa-frontend
$ npm install
$ bower install

> Edit the following file: ~/node_modules/awesome-typescript-loader/dist/instance.js
> Line ~142: Replace the body of the "applyDefaults" method with the following (from master):

function applyDefaults(configFilePath, compilerConfig, loaderConfig) {
    _.defaults(compilerConfig.options, {
        sourceMap: true,
        verbose: false,
        skipDefaultLibCheck: true,
        suppressOutputPathCheck: true
    });

    if (loaderConfig.transpileOnly) {
        compilerConfig.options.isolatedModules = true;
    }

    _.defaults(compilerConfig.options, {
        sourceRoot: compilerConfig.options.sourceMap ? process.cwd() : undefined
    });

    _.defaults(loaderConfig, {
        sourceMap: true,
        verbose: false,
    });

    delete compilerConfig.options.outDir;
    delete compilerConfig.options.inlineSourceMap;
    delete compilerConfig.options.outFile;
    delete compilerConfig.options.out;
    delete compilerConfig.options.noEmit;
}

Run project

For Development:

$ npm start

http://127.0.0.1:7777

OR

For Production:

$ npm run build

Create a virtual host and move the <<dist>> folder to the desired directory.
